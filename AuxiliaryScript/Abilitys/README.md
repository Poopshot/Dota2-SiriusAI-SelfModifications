## 技能模块
### 说明
技能模块，可自定义英雄技能策略。
### 命名规则
模块脚本命名以技能内部名称命名
### 修改要求
1. 技能释放功能中释放需遵从技能类型修改函数
2. 技能释放功能中无目标技能需去除castTarget ~= nil的判断
3. 获取范围不可超过1600
4. 建议在文件开头写明技能、释放键位、技能类型，不写也不影响文件
